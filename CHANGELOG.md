# Changelog

### Added

- Backend NestJS 11 con configuración base, validación global, CORS y Helmet.
- Conexión a MongoDB Atlas mediante Mongoose.
- Autenticación JWT con registro, login y ruta de perfil.
- Persistencia de usuarios con contraseña hasheada.
- Cuatro endpoints de formularios de proyecto: software, ecommerce, research e infrastructure.
- Validación específica por tipo de formulario.
- Documentación de variables de entorno, contrato de API, MongoDB Atlas y flujo Git por ramas.
- Ruta técnica `GET /api/health` para verificar el servidor en local.
- Guía de pruebas locales con navegador y curl.
